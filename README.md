## FKickAll

<p align="center">
  A Simple Plugin to Kick All Players Out with or without reason.
</p>

## Usage
```yaml
#Commands
/kickall <reason>
#Aliases
/fk <reason>
```
## Permission
```yaml
permission: kickall.use
```
